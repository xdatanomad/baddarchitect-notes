# Content Production Workflow

## 1. Intake

Start from raw notes in `content/notes/` or drafts in `content/articles/`.

Capture:

- Target reader.
- Reader problem.
- Search intent.
- Content pillar.
- Current source notes.
- Missing research or citations.

## 2. Brief

Create a short brief before drafting:

- Working title.
- One-sentence thesis.
- Outline.
- Examples or diagrams needed.
- SEO title and description draft.
- Internal links.

## 3. Draft

Write in a direct, practical voice. Avoid generic motivational language. Prefer concrete examples, checklists, and implementation guidance.

## 4. Review

Run the content quality checklist. For security or agent architecture topics, also run the AI security checklist.

## 5. Publish Prep

Add metadata, slug, summary, tags, and related links. Keep raw notes separate from publishable content.
